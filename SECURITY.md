# Security Policy

Do not open public issues that contain UFI device tokens, admin passwords,
OpenList tokens, EasyTier network secrets, private URLs, or screenshots showing
those values.

For public GitHub repositories, use GitHub Security Advisories when reporting a
private vulnerability. If advisories are not enabled, contact the maintainer
through a private channel before publishing exploit details.

The app stores settings in Android app-private storage. Treat debug APKs as
developer artifacts and do not distribute builds that include local signing
material or hardcoded service credentials.
